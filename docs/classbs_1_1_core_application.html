CoreApplication Class Reference
Layers » Core » Application

Description

Represents the primary entry point for the core systems.

Handles start-up, shutdown, primary loop and allows you to load and unload plugins.

Note
Sim thread only.

Inherits Module< CoreApplication >.

◆ runMainLoop()

void runMainLoop ( )

Executes the main loop.

This will update your components and modules, queue objects for rendering and run the simulation. Usually called immediately after startUp().

This will run infinitely until stopMainLoop is called (usually from another thread or internally).
