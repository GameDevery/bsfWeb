Description

Vulkan swap chain containing two or more buffers for rendering and presenting onto the screen.

Inherits INonCopyable.

◆ acquireBackBuffer()

void acquireBackBuffer ( )

Acquires a new back buffer image.

Caller can retrieve the surface by calling getBackBuffer(). Caller must wait on the semaphore provided by the surface before rendering to it.

Note
Must only be called once in-between present() calls, or before the first present() call.
