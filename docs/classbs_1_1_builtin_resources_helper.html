Checks all files in the specified folder for modifications compared to the time stored in the timestamp file.

Timestamp file must have been saved using writeTimestamp(). Returns 0 if no changes, 1 if timestamp is out date, or 2 if timestamp doesn't exist. lastUpdateTime will contain the time stored in the timestamp, if it exist.

Iterates over all the provided entries and generates a list of flags that determine should the asset be imported or not.

This is done by comparing file modification dates with the last update time and/or checking if any dependencies require import.

Parameters
[in]entriesJSON array containing entries to iterate over.
[in]inputFolderFolder in which to look for the input files.
[in]lastUpdateTimeTimestamp of when the last asset import occurred.
[in]forceImportIf true, all entries will be marked for import.
[in]dependenciesOptional map of entries that map each entry in the entries array, to a list of dependencies. The dependencies will then also be checked for modifications and if modified the entry will be marked for reimport.
[in]dependencyFolderFolder in which dependeny files reside. Only relevant if dependencies is provided.
Returns
An array of the same size as the entries array, containing value true if an asset should be imported, or false otherwise.

Iterates over all entires in the provided json file, imports the files linked by the entries them and stores them in the corresponding folder.

Also registers the imported files in the provided manifest.

Parameters
[in]entriesJSON array containing the entries to parse, with each entry containing data determine by set ImportMode.
[in]importFlagsA set of import flags (one for each entry) that specify which entries need to be imported.
[in]inputFolderFolder in which to look for the input files.
[in]outputFolderFolder in which to store the imported resources.
[in]manifestManifest in which to register the imported resources in.
[in]modeMode that controls how are files imported.
[in]dependenciesOptional map that be updated with any dependencies the imported assets depend on.
[in]compressTrue if the imported asset should be compressed when saved to the disk.

◆ AssetType

enum AssetType
strong

Flags that can control asset import.

Enumerator
Normal 

No flags, just import asset as normal.

Each entry is expected to have an "UUID" and a "Path" field.

Sprite 

Assumes imported assets are textures.

Will generate sprite assets for each imported texture. Expects "TextureUUID", "Path" and "SpriteUUID" fields present in per-entry JSON.
