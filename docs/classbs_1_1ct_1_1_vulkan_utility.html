static UINT32 calcInterfaceBlockElementSizeAndOffset ( GpuParamDataType  type,
UINT32  arraySize,
UINT32 &  offset 
)
static

Calculates the size and alignment of a single element within a shader interface block using the std140 layout.

Parameters
[in]typeType of the element. Structs are not supported.
[in]arraySizeNumber of array elements of the element (1 if it's not an array).
[in,out]offsetCurrent location in some parent buffer at which the element should be placed at. If the location doesn't match the element's alignment, the value will be modified to a valid alignment. In multiples of 4 bytes.
Returns
Size of the element, in multiples of 4 bytes.

const VkImageSubresourceRange &  cutWith,
std::array< VkImageSubresourceRange, 5 > &  output,
UINT32 &  numAreas 
)
static

Subdivides an image subresource range by cutting it with another range.

If the ranges don't overlap, or the cutWith range completely overs the toCut range, the original toCut range is output.

Parameters
[in]toCutRange to cut.
[in]cutWithRange to cut with.
[out]outputPieces of the range that was cut.
[out]numAreasNumber of pieces in the output array.
