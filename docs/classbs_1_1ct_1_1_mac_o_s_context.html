◆ endCurrent()

void endCurrent ( )
overridevirtual

Deactivates the rendering context.

Normally called just before setCurrent is called on another context.

Implements GLContext.

◆ lock()

void lock ( )

Locks the context so it can safely be used across threads.

Should be called before performing any OpenGL action or direct operation on the context. When done unlock it via unlock().

◆ markAsDirty()
