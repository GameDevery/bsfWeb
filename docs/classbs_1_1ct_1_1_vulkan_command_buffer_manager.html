◆ getSyncSemaphores()

void getSyncSemaphores ( UINT32  deviceIdx,
UINT32  syncMask,
VulkanSemaphore **  semaphores,
UINT32 &  count 
)

Returns a set of command buffer semaphores depending on the provided sync mask.

Parameters
[in]deviceIdxIndex of the device to get the semaphores for.
[in]syncMaskMask that has a bit enabled for each command buffer to retrieve the semaphore for. If the command buffer is not currently executing, semaphore won't be returned.
[out]semaphoresList containing all the required semaphores. Semaphores are tightly packed at the beginning of the array. Must be able to hold at least BS_MAX_UNIQUE_QUEUES entries.
[out]countNumber of semaphores provided in the semaphores array.
