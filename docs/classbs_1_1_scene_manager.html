SceneManager Class Reference
Internals » Core » Scene

Description

Keeps track of all active SceneObjects and their components.

Keeps track of component state and triggers their events. Updates the transforms of objects as SceneObjects move.

Inherits Module< SceneManager >.

◆ registerNewSO()

void registerNewSO ( const HSceneObject &  node)
protected

Register a new node in the scene manager, on the top-most level of the hierarchy.

Parameters
[in]nodeNode you wish to add. It's your responsibility not to add duplicate or null nodes. This method won't check.
Note

After you add a node in the scene manager, it takes ownership of its memory and is responsible for releasing it. Do NOT add nodes that have already been added (if you just want to change their parent). Normally this method will only be called by SceneObject.
