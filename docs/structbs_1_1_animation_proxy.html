Rebuilds the internal proxy data according to the newly assigned skeleton and clips.

This should be called whenever the animation skeleton changes.

Parameters
[in]skeletonNew skeleton to assign to the proxy.
[in]maskMask that filters which skeleton bones are enabled or disabled.
[in,out]clipInfosPotentially new clip infos that will be used for rebuilding the proxy. Once the method completes clip info layout and state indices will be populated for further use in the update*() methods.
[in]sceneObjectsA list of scene objects that are influenced by specific animation curves.
[in]morphShapesMorph shapes used for per-vertex animation.
Note
Should be called from the sim thread when the caller is sure the animation thread is not using it.

Rebuilds the internal proxy data according to the newly clips.

This should be called whenever clips are added or removed, or clip layout indices change.

Parameters
[in,out]clipInfosNew clip infos that will be used for rebuilding the proxy. Once the method completes clip info layout and state indices will be populated for further use in the update*() methods.
[in]sceneObjectsA list of scene objects that are influenced by specific animation curves.
[in]morphShapesMorph shapes used for per-vertex animation.
Note
