Exception Class Reference
Layers » Utility » Error handling

Description

Base class for all Banshee exceptions.

Note
Banshee doesn't perform exception handling, but these classes remain here in case others wish to enable them.

Inherits exception.

Inherited by FileNotFoundException, InternalErrorException, InvalidParametersException, InvalidStateException, IOException, NotImplementedException, RenderingAPIException, and UnitTestException.
