DeferredIBLProbeMat Class Reference

Description

Shader that renders an individual reflection probe for image based lighting.

This is an alternative to TiledDeferredImageBasedLighting for cases when compute shaders are not usable or suitable. Must be preceeded by DeferredIBLSetupMat and followed by DeferredIBLSkyMat and DeferredIBLFinalizeMat.

Inherits RendererMaterial< DeferredIBLProbeMat >.
