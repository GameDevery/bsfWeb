static SPtr<IndexBuffer> create ( const INDEX_BUFFER_DESC &  desc,
GpuDeviceFlags  deviceMask = GDF_DEFAULT 
)
static

Creates a new index buffer that holds indices referencing vertices in a vertex buffer.

Indices are interpreted by the pipeline and vertices are drawn in the order specified in the index buffer.

Parameters
[in]descDescription of the buffer to create.
[in]deviceMaskMask that determines on which GPU devices should the object be created on.
