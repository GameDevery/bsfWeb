Parses the provided object, serializes all of its data as specified by its RTTIType and returns the data in the form of raw memory.

Parameters
[in]objectObject to encode.
[in]bytesWrittenOutput value containing the total number of bytes it took to encode the object.
[in]allocatorDetermines how is memory allocated. If not specified the default allocator is used.
[in]shallowDetermines how to handle referenced objects. If true then references will not be encoded and will be set to null. If false then references will be encoded as well and restored upon decoding.
[in]paramsOptional parameters to be passed to the serialization callbacks on the objects being serialized.
Returns
A buffer containing the encoded object. It is up to the user to release the buffer memory when no longer needed.
