◆ findBestVisual()

GLVisualConfig findBestVisual ( ::Display *  display,
bool  depthStencil,
UINT32  multisample,
bool  srgb 
) const

Selects an appropriate X11 visual info depending on the provided parameters.

Visual info should then be used for creation of an X11 window.

Parameters
[in]displayX11 display the window will be created on.
[in]depthStencilTrue if the window requires a depth-stencil buffer.
[in]multisampleNumber of samples per pixel, if window back buffer requires support for multiple samples. Set to 0 or 1 if multisampling is not required.
[in]srgbIf enabled the pixels written to the back-buffer are assumed to be in linear space and will automatically be encoded into gamma space on write.
Returns
X11 visual info structure you may use to initialize a window.
