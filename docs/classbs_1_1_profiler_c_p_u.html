ProfilerCPU Class Reference
Layers » Core » Profiling

Description

Provides various performance measuring methods.

Note
Thread safe. Matching begin* \ end* calls must belong to the same thread though.

Inherits Module< ProfilerCPU >.

◆ beginSamplePrecise()

void beginSamplePrecise ( const char *  name)

Begins precise sample measurement.

Must be followed by endSamplePrecise().

Parameters
[in]nameUnique name for the sample you can later use to find the sampling data.
Note

This method uses very precise CPU counters to determine variety of data not provided by standard beginSample(). However due to the way these counters work you should not use this method for larger parts of code. It does not consider context switches so if the OS decides to switch context between measurements you will get invalid data.
