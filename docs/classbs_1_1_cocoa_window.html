CocoaWindow Class Reference
Internals » Utility | Internal-Utility » Platform

Description

Represents a Cocoa window.

Note this class should only be used from the sim thread as Cocoa does not support event handling, windows or views outside of the main thread.

◆ _setDragZones()

void _setDragZones ( const Vector< Rect2I > &  rects)

Sets a portion of the window in which the user can click and drag in order to move the window.

This is needed when window has no title bar, yet you still want to allow the user to drag it by clicking on some specific area (e.g. a title bar you manually render).

Parameters
[in]rectsAreas of the window (relative to the window origin in top-left corner) in which the drag operation in allowed.
