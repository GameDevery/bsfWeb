◆ blit()

void blit ( const SPtr< Texture > &  texture,
const Rect2I &  area = Rect2I::EMPTY,
bool  flipUV = false,
bool  isDepth = false 
)

Blits contents of the provided texture into the currently bound render target.

If the provided texture contains multiple samples, they will be resolved.

Parameters
[in]textureSource texture to blit.
[in]areaArea of the source texture to blit in pixels. If width or height is zero it is assumed the entire texture should be blitted.
[in]flipUVIf true, vertical UV coordinate will be flipped upside down.
[in]isDepthIf true, the input texture is assumed to be a depth texture (instead of a color one). Multisampled depth textures will be resolved by taking the minimum value of all samples, unlike color textures which wil be averaged.

void drawScreenQuad ( const Rect2 &  uv,
const Vector2I &  textureSize = Vector2I(1, 1),
UINT32  numInstances = 1,
bool  flipUV = false 
)

Draws a quad over the entire viewport in normalized device coordinates.

Parameters
[in]uvUV coordinates to assign to the corners of the quad.
[in]textureSizeSize of the texture the UV coordinates are specified for. If the UV coordinates are already in normalized (0, 1) range then keep this value as is. If the UV coordinates are in texels then set this value to the texture size so they can be normalized internally.
[in]numInstancesHow many instances of the quad to draw (using instanced rendering). Useful when drawing to 3D textures.
[in]flipUVIf true, vertical UV coordinate will be flipped upside down.
Note
Core thread.
