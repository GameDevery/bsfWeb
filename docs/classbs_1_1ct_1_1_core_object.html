CoreObject Class Reference
Layers » Core » Core thread

Description

Represents counterpart of a CoreObject that is meant to be used specifically on the core thread.

Note
Core thread only.
Different CoreObject implementations should implement this class for their own needs.

Inherited by BlendState, Camera, ComputePipelineState, DepthStencilState, GpuBuffer, GpuParamBlockBuffer, GpuParams, GpuPipelineParamInfo, GpuProgram, GraphicsPipelineState, IndexBuffer, Light, LightProbeVolume, Material, MeshBase, MeshHeap, ParticleSystem, Pass, RasterizerState, ReflectionProbe, Renderable, RenderTarget, SamplerState, Shader, Skybox, Technique, Texture, VertexBuffer, VertexDeclaration, and Viewport.

Blocks the current thread until the resource is fully initialized.

Note
