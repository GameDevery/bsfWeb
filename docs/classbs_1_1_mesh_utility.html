Calculates per-vertex normals based on the provided vertices and indices.

Parameters
[in]verticesSet of vertices containing vertex positions.
[in]indicesSet of indices containing indexes into vertex array for each triangle.
[in]numVerticesNumber of vertices in the vertices array.
[in]numIndicesNumber of indices in the indices array. Must be a multiple of three.
[out]normalsPre-allocated buffer that will contain the calculated normals. Must be the same size as the vertex array.
[in]indexSizeSize of a single index in the indices array, in bytes.
Note

Vertices should be split before calling this method if there are any discontinuities. (for example a vertex on a corner of a cube should be split into three vertices used by three triangles in order for the normals to be valid.)

Calculates per-vertex tangents and bitangents based on the provided vertices, uv coordinates and indices.

Parameters
[in]verticesSet of vertices containing vertex positions.
[in]normalsSet of normals to use when calculating tangents. Must the the same length as the number of vertices.
[in]uvSet of UV coordinates to use when calculating tangents. Must the the same length as the number of vertices.
[in]indicesSet of indices containing indexes into vertex array for each triangle.
[in]numVerticesNumber of vertices in the vertices, normals and uv arrays.
[in]numIndicesNumber of indices in the indices array. Must be a multiple of three.
[out]tangentsPre-allocated buffer that will contain the calculated tangents. Must be the same size as the vertex array.
[out]bitangentsPre-allocated buffer that will contain the calculated bitangents. Must be the same size as the vertex array.
[in]indexSizeSize of a single index in the indices array, in bytes.
[in]vertexStrideNumber of bytes to advance the vertices, normals and uv arrays with each vertex. If set to zero them each array is advanced according to its own size.
Note

Vertices should be split before calling this method if there are any discontinuities. (for example a vertex on a corner of a cube should be split into three vertices used by three triangles in order for the normals to be valid.)

)
static

Clips a set of two-dimensional vertices and uv coordinates against a set of arbitrary planes.

Parameters
[in]verticesA set of vertices in Vector2 format. Each vertex should be vertexStride bytes from each other.
[in]uvsA set of UV coordinates in Vector2 format. Each coordinate should be vertexStride bytes from each other. Can be null if UV is not needed.
[in]numTrisNumber of triangles to clip (must be number of vertices/uvs / 3).
[in]vertexStrideDistance in bytes between two separate vertex or UV values in the provided vertices and uvs buffers.
[in]clipPlanesA set of planes to clip the vertices against. Since the vertices are two-dimensional the plane's Z coordinate should be zero.
[in]writeCallbackCallback that will be triggered when clipped vertices and UV coordinates are generated and need to be stored. Vertices are always generate in tuples of three, forming a single triangle.
