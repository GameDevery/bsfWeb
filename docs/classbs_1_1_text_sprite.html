Calculates text quads you may use for text rendering, based on the specified text data.

Only generates quads for the specified page.

Parameters
[in]pageFont page to generate the data for.
[in]textDataText data to generate offsets for.
[in]widthWidth of the text bounds into which to constrain the text, in pixels.
[in]heightHeight of the text bounds into which to constrain the text, in pixels.
[in]horzAlignSpecifies how is text horizontally aligned within its bounds.
[in]vertAlignSpecifies how is text vertically aligned within its bounds.
[in]anchorDetermines how to anchor the text within the bounds.
[out]verticesOutput buffer containing quad positions. Must be allocated and of adequate size.
[out]uvOutput buffer containing quad UV coordinates. Must be allocated and of adequate size. Can be null.
[out]indicesOutput buffer containing quad indices. Must be allocated and of adequate size. Can be null.
[in]bufferSizeQuadsSize of the output buffers, in number of quads.
Returns
Number of generated quads.

Calculates and returns offset for each individual text line.

The offsets provide information on how much to offset the lines within provided bounds.

Parameters
[in]textDataText data to generate offsets for.
[in]widthWidth of the text bounds into which to constrain the text, in pixels.
[in]heightHeight of the text bounds into which to constrain the text, in pixels.
[in]horzAlignSpecifies how is text horizontally aligned within its bounds.
[in]vertAlignSpecifies how is text vertically aligned within its bounds.
[out]outputPre-allocated buffer to output the results in. Buffer must have an element for every line in textData.
