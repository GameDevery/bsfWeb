TimerQuery Class Referenceabstract
Internals » Core » RenderAPI

Description

Represents a GPU query that measures execution time of GPU operations.

The query will measure any GPU operations that take place between its begin() and end() calls.

Note
Core thread only.

Inherited by D3D11TimerQuery, GLTimerQuery, and VulkanTimerQuery.

virtual void begin ( const SPtr< CommandBuffer > &  cb = nullptr)
pure virtual

Starts the counter.

Parameters
[in]cbOptional command buffer to queue the operation on. If not provided operation is executed on the main command buffer. Otherwise it is executed when RenderAPI::executeCommands() is called. Buffer must support graphics or compute operations.
Note
Place any commands you want to measure after this call. Call "end" when done.

Implemented in D3D11TimerQuery, GLTimerQuery, and VulkanTimerQuery.
