RenderWindow Class Referenceabstract
Layers » Core » RenderAPI

Description

Operating system window with a specific position, size and style.

Each window serves as a surface that can be rendered into by RenderAPI operations.

Inherits RenderTarget.

Inherited by D3D11RenderWindow, LinuxRenderWindow, LinuxRenderWindow, MacOSRenderWindow, Win32RenderWindow, and Win32RenderWindow.

Switches the window to fullscreen mode.

Child windows cannot go into fullscreen mode.

Parameters
[in]widthWidth of the window frame buffer in pixels.
[in]heightHeight of the window frame buffer in pixels.
[in]refreshRateRefresh rate of the window in Hertz.
[in]monitorIdxIndex of the monitor to go fullscreen on.
Note
If the exact provided mode isn't available, closest one is used instead.
This is an asynchronous method.
