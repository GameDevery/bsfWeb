RenderCompositor Class Reference
Plugins » RenderBeast

Description

Performs rendering by iterating over a hierarchy of render nodes.

Each node in the hierarchy performs a specific rendering tasks and passes its output to the dependant node. The system takes care of initializing, rendering and cleaning up nodes automatically depending on their dependencies.

Rebuilds the render node hierarchy.

Call this whenever some setting that may influence the render node dependencies changes.

Parameters
[in]viewParent view to which this compositor belongs to.
[in]finalNodeIdentifier of the final node in the node hierarchy. This node is expected to write to the views render target. All other nodes will be deduced from this node's dependencies.
