◆ getShiftedLightPosition()

Vector3 getShiftedLightPosition ( ) const

Calculates the light position that is shifted in order to account for area spot lights.

For non-spot lights this method will return normal light position. The position will be shifted back from the light direction, magnitude of the shift depending on the source radius.
