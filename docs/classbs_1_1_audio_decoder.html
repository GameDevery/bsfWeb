AudioDecoder Class Referenceabstract
Plugins » BansheeOpenAudio

Description

Interface used for implementations that parse audio formats into a set of PCM samples.

Inherited by FLACDecoder, OggVorbisDecoder, and WaveDecoder.

◆ seek()

Moves the read pointer to the specified offset.

Any further read() calls will read from this location. User must ensure not to seek past the end of the data.

Parameters
