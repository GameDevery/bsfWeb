const Map< UINT32, SPtr< PixelData >> &  icons 
)
static

Updates icons in the provided executable.

Only icons that already exist in the executable can be updated, no new icons can be inserted. Icons must be square.

Parameters
[in]filePathPath to the executable.
[in]iconsPixels of images to replace. Each entry maps an icon width (and height, since they're square) to its pixels.
