GroupAlloc Class Reference
Internals » Utility | Internal-Utility » Memory

Description

Provides an easy way to group multiple allocations under a single (actual) allocation.

Requires the user to first call reserve() methods for all requested data elements, followed by init(), after which allocation/deallocation can follow using construct/destruct or alloc/free methods.

Inherits INonCopyable.
