GUIPanel Class Reference
Layers » Engine » GUI

Description

Represents a GUI panel that you can use for free placement of GUI elements within its bounds.

Inherits GUILayout.

◆ create() [1/3]

UINT16  depthRangeMax = -1 
)
static

Creates a new GUI panel.

Parameters
[in]depthDetermines rendering order of the GUI panel. Panels with lower depth will be rendered in front of panels with higher depth. Provided depth is relative to depth of the parent GUI panel (if any).
[in]depthRangeMinMinimum range of depths that children of this GUI panel can have. If any panel has depth outside of the range [depth - depthRangeMin, depth + depthRangeMax] it will be clamped to nearest extreme. Value of -1 means infinite range.
[in]depthRangeMaxMaximum range of depths that children of this GUI panel can have. If any panel has depth outside of the range [depth - depthRangeMin, depth + depthRangeMax] it will be clamped to nearest extreme. Value of -1 means infinite range.
