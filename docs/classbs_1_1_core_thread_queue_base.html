◆ submitToCoreThread()

void submitToCoreThread ( bool  blockUntilComplete = false)

Makes all the currently queued commands available to the core thread.

They will be executed as soon as the core thread is ready. All queued commands are removed from the queue.

Parameters
[in]blockUntilCompleteIf true, the calling thread will block until the core thread finished executing all currently queued commands. This is usually very expensive and should only be used in non-performance critical code.
