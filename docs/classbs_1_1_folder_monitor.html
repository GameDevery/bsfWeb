FolderMonitor Class Reference
Internals » Core » Platform

Description

Allows monitoring a file system folder for changes.

Depending on the flags set this monitor can notify you when file is changed/moved/renamed and similar.

)

Starts monitoring a folder at the specified path.

Parameters
[in]folderPathAbsolute path to the folder you want to monitor.
[in]subdirectoriesIf true, provided folder and all of its subdirectories will be monitored for changes. Otherwise only the provided folder will be monitored.
[in]changeFilterA set of flags you may OR together. Different notification events will trigger depending on which flags you set.
