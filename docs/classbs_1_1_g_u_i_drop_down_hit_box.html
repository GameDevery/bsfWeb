GUIDropDownHitBox Class Reference
Internals » Engine » GUI

Description

Helper class used for detecting when a certain area is in focus, and getting notified when that state changes.

Inherits GUIElementContainer.

)
static

Creates a new drop down hit box that will detect mouse input over certain area.

You must call setBounds() to define the area.

Parameters
[in]captureMouseOverIf true mouse over/out/move events will be captured by this control and wont be passed to other GUI elements.
[in]captureMousePressesIf true mouse clicks will be captured by this control and wont be passed to other GUI elements.
