◆ preintegratedEnvGF

SPtr<Texture> preintegratedEnvGF
static

2D 2-channel texture containing a pre-integrated G and F factors of the microfactet BRDF.

This is an approximation used for image based lighting, so we can avoid sampling environment maps for each light. Works in tandem with the importance sampled reflection cubemaps.

(u, v) = (NoV, roughness) (r, g) = (scale, bias)
