T(&)  coefficients[4] 
)
static

Calculates coefficients needed for evaluating a cubic curve in Hermite form.

Assumes t has been normalized is in range [0, 1]. Tangents must be scaled by the length of the curve (length is the maximum value of t before it was normalized).

Parameters
[in]pointAStarting point (at t=0).
[in]pointBEnding point (at t=1).
[in]tangentAStarting tangent (at t=0).
[in]tangentBEnding tangent (at t = 1).
[out]coefficientsFour coefficients for the cubic curve, in order [t^3, t^2, t, 1].

Calculates coefficients needed for evaluating a cubic curve in Hermite form.

Assumes t is in range [0, length]. Tangents must not be scaled by length.

Parameters
[in]pointAStarting point (at t=0).
[in]pointBEnding point (at t=length).
[in]tangentAStarting tangent (at t=0).
[in]tangentBEnding tangent (at t=length).
[in]lengthMaximum value the curve will be evaluated at.
[out]coefficientsFour coefficients for the cubic curve, in order [t^3, t^2, t, 1].
