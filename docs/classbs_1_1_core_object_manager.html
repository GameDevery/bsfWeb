◆ syncToCore() [1/2]

void syncToCore ( )

Synchronizes all dirty CoreObjects with the core thread.

Their dirty data will be allocated using the global frame allocator and then queued for update using the core thread queue for the calling thread.

Note
Sim thread only.
This is an asynchronous method.
