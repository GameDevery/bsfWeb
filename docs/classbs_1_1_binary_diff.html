BinaryDiff Class Reference
Internals » Utility | Internal-Utility » Serialization

Description

Generates and applies "diffs".

Diffs contain per-field differences between an original and new object. These differences can be saved and then applied to an original object to transform it to the new version.

Note
Objects must be in intermediate serialized format generated by BinarySerializer.

Inherits IDiff.
