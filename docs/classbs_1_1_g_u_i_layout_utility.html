Parameters

[in]layoutParent layout of the children to calculate the area for.
[in]updateOptimalSizesOptimization (doesn't change the results). Set to false if GUIElementBase::_updateOptimalLayoutSizes was already called and optimal sizes are up to date to avoid recalculating them. (Normally that is true if this is being called during GUI layout update)
