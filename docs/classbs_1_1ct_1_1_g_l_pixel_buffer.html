GLPixelBuffer Class Reference
Plugins » BansheeGLRenderAPI

Description

Represents a hardware buffer that stores a single pixel surface.

This may be a 1D, 2D or 3D surface, but unlike a texture it consists only of a single surface (no mip maps, cube map faces or similar).

Inherited by GLTextureBuffer.
