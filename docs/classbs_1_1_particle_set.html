ParticleSet Class Reference
Internals » Core » Particles

Description

Provides a simple and fast way to allocate and deallocate particles.

Inherits INonCopyable.

Method documentation

◆ allocParticles()

UINT32 allocParticles ( UINT32  count)

Allocates a number of new particles and returns the index to the particle.

Note that the returned index is not persistent and can become invalid after a call to freeParticle(). Returns the index to the first allocated particle.

◆ clear()

void clear ( UINT32  numPartices = 0)

Frees all active partices past the provided particle count (0 to clear all particles).

◆ determineTextureSize()
