VideoMode Class Reference
Layers » Core » RenderAPI

Description

Video mode contains information about how a render window presents its information to an output device like a monitor.

Inherited by D3D11VideoMode, LinuxVideoMode, LinuxVideoMode, MacOSVideoMode, Win32VideoMode, and Win32VideoMode.
