StringID Class Reference
Layers » Utility » String

Description

A string identifier that provides very fast comparisons to other string identifiers.

Significantly faster than comparing raw strings.

Note

Essentially a unique ID is generated for each string and then the ID is used for comparisons as if you were using an integer or an enum.
Thread safe.
