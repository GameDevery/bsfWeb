◆ _enableAllWindows()

static void _enableAllWindows ( )
static

Enables all open windows.

Enabled windows can receive mouse and keyboard input. This includes even windows disabled because there is a modal window on top of them.

◆ _restoreModalWindows()

static void _restoreModalWindows ( )
static

Restores disabled state of all windows that were disabled due to modal windows being on top of them.

Companion method to _enableAllWindows() that can help restore original state after it is called.
