GaussianDOFCombineMat Class Reference
Plugins » RenderBeast

Description

Shader that combines pixels for near unfocused, focused and far unfocused planes, as calculated by GaussianDOFSeparateMat.

Outputs final depth-of-field filtered image.

Inherits RendererMaterial< GaussianDOFCombineMat >.

Renders the post-process effect with the provided parameters.

Parameters
[in]focusedInput texture containing focused (default) scene color.
[in]nearInput texture containing filtered (blurred) values for the unfocused foreground area. Can be null if no near plane needs to be blended.
[in]farInput texture containing filtered (blurred) values for the unfocused background area. Can be null if no far plane needs to be blended.
[in]depthInput depth buffer texture that will be used for determining pixel depth.
[in]outputTexture to output the results to.
[in]viewView through which the depth of field effect is viewed.
[in]settingsSettings used to control depth of field rendering.
