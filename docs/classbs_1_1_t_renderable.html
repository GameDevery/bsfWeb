[IMPLEMENTATION]

Description

template<bool Core>
class bs::TRenderable< Core >

Renderable represents any visible object in the scene.

It has a mesh, bounds and a set of materials. Renderer will render any Renderable objects visible by a camera.

Inherits SceneActor.
