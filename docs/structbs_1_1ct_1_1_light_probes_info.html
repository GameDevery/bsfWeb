◆ tetrahedra

SPtr<GpuBuffer> tetrahedra

Contains information about tetrahedra formed by light probes.

First half of the buffer is populated by actual tetrahedrons, while the second half is populated by information about outer faces (triangles). numTetrahedra marks the spot where split happens.

◆ tetrahedraVolume

SPtr<Mesh> tetrahedraVolume

Mesh representing the entire light probe volume.

Each vertex has an associated tetrahedron (or face) index which can be used to map into the tetrahedra array to retrieve probe information.
