void setCursorIcon ( const String &  name,
const PixelData &  pixelData,
const Vector2I &  hotSpot 
)

Registers a new custom cursor icon you can then set by calling "setCursor".

Parameters
[in]nameThe name to identify the cursor.
[in]pixelDataCursor image data.
[in]hotSpotOffset on the cursor image to where the actual input happens (for example tip of the Arrow cursor).
Note

Stores an internal copy of the pixel data. Clear it by calling removeCursorIcon(). If a custom icon with the same name already exists it will be replaced.
