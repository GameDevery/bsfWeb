MorphShape Class Reference
Internals » Core » Animation

Description

A set of vertices representing a single shape in a morph target animation.

Vertices are represented as a difference between base and target shape.

Inherits IReflectable.

Creates a new morph shape from the provided set of vertices.

Parameters
[in]nameName of the frame. Must be unique within a morph channel.
[in]weightWeight in range [0, 1]. Determines how are sequential shapes animated between within a morph channel. e.g. if there is a shape A with weight 0.3 and shape B with weight 0.8 then shape A will be displayed first and then be blended towards shape B as time passes.
[in]verticesVertices of the base mesh modified by the shape.
