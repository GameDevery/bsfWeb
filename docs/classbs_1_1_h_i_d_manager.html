HIDManager Class Reference

Description

Provides access to the low level IO HID manager.

Enumerates available input devices and reports their input to the Input object.

Checks if any new input events have been generates and reports them to the Input object.

Parameters
[in]deviceDevice to read events from. If null, the events are read from all devices of the compatible type.
[in]ignoreEventsIf true the system will not trigger any external events for the reported input. This can be useful for situations where input is disabled, like an out-of-focus window.
