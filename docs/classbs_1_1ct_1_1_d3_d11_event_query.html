D3D11EventQuery Class Reference
Plugins » BansheeD3D11RenderAPI

Description

Represents a GPU query that gets triggered when GPU starts processing the query.

Note

Normally GPU will have many commands in its command buffer. When begin() is called it is placed in that command buffer. Once the buffer empties and GPU reaches the EventQuery command, the query callback is triggered.

Core thread only.

Inherits EventQuery.

Parameters
[in]cbOptional command buffer to queue the operation on. If not provided operation is executed on the main command buffer. Otherwise it is executed when RenderAPI::executeCommands() is called. Buffer must support graphics or compute operations.
Note

Once the query is started you may poll isReady() method to check when query has finished, or you may hook up an onTriggered callback and be notified that way.
