Gets internal Vulkan framebuffer object.

Parameters
[in]loadMaskMask that control which render target surface contents should be preserved on load.
[in]readMaskMask that controls which render targets can be read by shaders while they're bound.
[in]clearMaskMask that controls which render targets should be cleared on render pass start. Target cannot have both load and clear bits set. If load bit is set, clear will be ignored.
