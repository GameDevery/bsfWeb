Creates a new GPU program using the provided source code.

If compilation fails or program is not supported isCompiled() with return false, and you will be able to retrieve the error message via getCompileErrorMessage().

Parameters
[in]descDescription of the program to create.
[in]deviceMaskMask that determines on which GPU devices should the object be created on.
