SpriteMaterial Class Reference
Internals » Engine » 2D

Description

Interfaced implemented by materials used for rendering sprites.

This is expected to be used as a singleton.

Inherited by SpriteImageOpaqueMaterial, SpriteImageTransparentMaterial, SpriteLineMaterial, and SpriteTextMaterial.

virtual

Generates a hash value that describes the contents of the sprite material info structure.

Returned hash doesn't guarantee that the two objects with the same hash are idential, but rather that the objects are mergeable via merge().

virtual void merge ( SpriteMaterialInfo &  mergeInto,
const SpriteMaterialInfo &  mergeFrom 
) const
virtual

Merges two SpriteMaterialInfos into one structure.

User must guarantee that the two objects are mergeable by ensuring their merge hashes match (by calling getMergeHash()).

Parameters
[in,out]mergeIntoObject that contains the first part of the data, and will contain the result of the merge.
[in]mergeFromObject that contains the second part of the data to merge, which will be merged into the first object.
