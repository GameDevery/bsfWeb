◆ lockWindows()

void lockWindows ( )
static

Locks the access to the list of CocoaWindows ensuring no new windows can be created or destroyed.

This must be called any time CocoaWindow is being used outside of the main thread to ensure the window doesn't get destroyed while being in use. Needs to be followed by unlockWindows().
