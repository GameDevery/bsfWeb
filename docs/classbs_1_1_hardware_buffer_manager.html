SPtr<GpuBuffer> createGpuBuffer ( const GPU_BUFFER_DESC &  desc)

Creates a generic buffer that can be passed as a parameter to a GPU program.

This type of buffer can hold various type of data and can be used for various purposes. See GpuBufferType for explanation of different buffer types.

Parameters

◆ createGpuParamBlockBuffer()

SPtr<GpuParamBlockBuffer> createGpuParamBlockBuffer ( UINT32  size,
GpuParamBlockUsage  usage = GPBU_DYNAMIC 
)

Creates an GPU parameter block that you can use for setting parameters for GPU programs.

Parameter blocks may be used for sharing parameter data between multiple GPU programs, requiring you to update only one buffer for all of them, potentially improving performance.

Parameters
[in]sizeSize of the parameter buffer in bytes.
[in]usageUsage that tells the hardware how will be buffer be used.

SPtr<IndexBuffer> createIndexBuffer ( const INDEX_BUFFER_DESC &  desc)

Creates a new index buffer that holds indices referencing vertices in a vertex buffer.

Indices are interpreted by the pipeline and vertices are drawn in the order specified in the index buffer.

[in]descDescription of the buffer to create.
