Converts a set of audio samples of a certain bit depth to a set of floating point samples in range [-1, 1].

Parameters

[in]inBitDepthSize of a single sample in the input array, in bits.
[out]outputPre-allocated buffer to store the output samples in. Total size of the buffer should be numSamples * sizeof(float).
[in]numSamplesTotal number of samples to process.

static

Converts a set of audio samples using multiple channels into a set of mono samples.

Parameters
[in]inputA set of input samples. Per-channels samples should be interleaved. Size of each sample is determined by bitDepth. Total size of the buffer should be numSamples * numChannels * bitDepth / 8.
[out]outputPre-allocated buffer to store the mono samples. Should be of numSamples * bitDepth / 8 size.
[in]bitDepthSize of a single sample in bits.
[in]numSamplesNumber of samples per a single channel.
[in]numChannelsNumber of channels in the input data.
