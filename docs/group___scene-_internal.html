◆ GameObjectHandleDeserializationMode

enum GameObjectHandleDeserializationMode

Possible modes to use when deserializing games objects.

Enumerator
GODM_UseOriginalIds 

All handles will point to old ID that were restored from the deserialized file.

GODM_UseNewIds 

All handles will point to new IDs that were given to the deserialized GameObjects.

GODM_RestoreExternal 

Handles pointing to GameObjects outside of the currently deserialized set will attempt to be restored in case those objects are still active.

GODM_BreakExternal 

Handles pointing to GameObjects outside of the currently deserialized set will be broken.

GODM_KeepMissing 

Handles pointing to GameObjects that cannot be found will not be set to null.

◆ TransformChangedFlags

enum TransformChangedFlags

Flags used for notifying child scene object and components when a transform has been changed.

Enumerator
TCF_None 

Component will not be notified about any events relating to the transform.

TCF_Transform 

Component will be notified when the its position, rotation or scale has changed.

TCF_Parent 

Component will be notified when its parent changes.

TCF_Mobility 

Component will be notified when mobility state changes.
