void initArray ( const String &  name,
UINT16  uniqueId,
Any  getter,
Any  getSize,
Any  setter,
Any  setSize,
UINT64  flags 
)

Initializes a field containing an array of pointers to data types implementing IReflectable interface.

Parameters
[in]nameName of the field.
[in]uniqueIdUnique identifier for this field. Although name is also a unique identifier we want a small data type that can be used for efficiently serializing data to disk and similar. It is primarily used for compatibility between different versions of serialized data.
[in]getterThe getter method for the field. Must be a specific signature: DataType*(ObjectType*, UINT32)
[in]getSizeGetter method that returns the size of an array. Must be a specific signature: UINT32(ObjectType*)
[in]setterThe setter method for the field. Must be a specific signature: void(ObjectType*, UINT32, DataType*)
[in]setSizeSetter method that allows you to resize an array. Can be null. Must be a specific signature: void(ObjectType*, UINT32)
[in]flagsVarious flags you can use to specialize how systems handle this field. See "RTTIFieldFlag".
