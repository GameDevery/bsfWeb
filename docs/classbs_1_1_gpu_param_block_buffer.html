GpuParamBlockBuffer Class Reference
Layers » Core » RenderAPI

Description

Represents a GPU parameter block buffer.

Parameter block buffers are bound to GPU programs which then fetch parameters from those buffers.

Writing or reading from this buffer will translate directly to API calls that update the GPU.

Note
Sim thread only.

Inherits CoreObject.

GpuParamBlockUsage  usage = GPBU_DYNAMIC 
)
static

Creates an GPU parameter block that you can use for setting parameters for GPU programs.

Parameter blocks may be used for sharing parameter data between multiple GPU programs, requiring you to update only one buffer for all of them, potentially improving performance.

Parameters
[in]sizeSize of the parameter buffer in bytes.
[in]usageUsage that tells the hardware how will be buffer be used.
