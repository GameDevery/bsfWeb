Enumeration Type Documentation

◆ AnimationCurveFlag

enum AnimationCurveFlag
strong

Flags that describe an animation curve.

Enumerator
ImportedCurve 

If enabled, the curve was imported from an external file and not created within the engine.

This will affect how are animation results applied to scene objects (with imported animations it is assumed the curve is animating bones and with in-engine curves it is assumed the curve is animating scene objects).

MorphFrame 

Signifies the curve is used to animate between different frames within a morph channel.

In range [0, 1].

MorphWeight 

Signifies the curve is used to adjust the weight of a morph channel.

In range [0, 1].
