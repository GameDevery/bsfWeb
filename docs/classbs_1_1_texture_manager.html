TextureManager Class Referenceabstract
Internals » Core » Resources

Description

Defines interface for creation of textures.

Render systems provide their own implementations.

Note
Sim thread only.

Inherits Module< TextureManager >.

Inherited by D3D11TextureManager, GLTextureManager, and VulkanTextureManager.
