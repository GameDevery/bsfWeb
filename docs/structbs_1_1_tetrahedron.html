◆ neighbors

INT32 neighbors[4]

Indices pointing to neighbor tetrahedrons.

Each neighbor index maps to the vertices array, so neighbor/vertex pair at the same location will be the only neighbor not containing that vertex (i.e. neighbor opposite to the vertex). If a tetrahedron is on the volume edge, it has only three neighbors and its last neighbor will be set to -1.
