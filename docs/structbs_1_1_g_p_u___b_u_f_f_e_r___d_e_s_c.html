Member Data Documentation

◆ elementCount

UINT32 elementCount

Number of elements in the buffer.

◆ elementSize

UINT32 elementSize

Size of each individual element in the buffer, in bytes.

Only needed if using non-standard buffer. If using standard buffers element size is calculated from format and this must be zero.

◆ format

GpuBufferFormat format

Format if the data in the buffer.

Only relevant for standard buffers, must be BF_UNKNOWN otherwise.

◆ randomGpuWrite

bool randomGpuWrite = false

When true allows the GPU to write to the resource.

Must be enabled if buffer type is GBT_APPENDCONSUME.

◆ type

GpuBufferType type

Type of the buffer.

Determines how is buffer seen by the GPU program and in what ways can it be used.

◆ usage

GpuBufferUsage usage = GBU_STATIC

Usage that tells the hardware how will be buffer be used.

◆ useCounter

bool useCounter = false

When true binds a counter that can be used from a GPU program on the buffer.

Can only be used in combination with GBT_STRUCTURED and randomGpuWrite must be enabled.
