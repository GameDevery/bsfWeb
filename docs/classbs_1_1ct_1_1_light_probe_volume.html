◆ renderProbes()

bool renderProbes ( UINT32  maxProbes)
protected

Renders dirty probes and updates their SH coefficients in the local GPU buffer.

Parameters
[in]maxProbesMaximum number of probes to render. Set to zero to render all dirty probes. Limiting the number of probes allows the rendering to be distributed over multiple frames.
Returns
True if there are no more dirty probes to process.
