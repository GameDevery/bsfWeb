◆ open()

bool open ( std::function< void(UINT8 *, UINT32)>  writeCallback,
UINT32  sampleRate,
UINT32  bitDepth,
UINT32  numChannels 
)

Sets up the writer.

Should be called before calling write().

Parameters
[in]writeCallbackCallback that will be triggered when the writer is ready to output some data. The callback should copy the provided data into its own buffer.
[in]sampleRateDetermines how many samples per second the written data will have.
[in]bitDepthDetermines the size of a single sample, in bits.
[in]numChannelsDetermines the number of audio channels. Channel data will be output interleaved in the output buffer.
