◆ buildMeshes()

Vector<ShapeMeshData> buildMeshes ( SortType  sorting = SortType::None,
const Vector3 &  reference = Vector3::ZERO,
UINT64  layers = 0xFFFFFFFFFFFFFFFF 
)

Generates a set of meshes from all the recorded solid and wireframe shapes.

The meshes can be accessed via getMeshes() and released via clearMeshes().

Parameters
sorting(optional) Determines how (and if) should elements be sorted based on their distance from the reference point.
reference(optional) Reference point to use for determining distance when sorting.
layers(optional) Layers bitfield that can be used for controlling which shapes will be included in the mesh. This bitfield will be ANDed with the layer specified when recording the shape.
Returns
Generated mesh data.
