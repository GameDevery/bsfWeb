Finds an existing or creates a new D3D11 input layout.

Input layout maps a vertex declaration from a vertex buffer to vertex program input declaration

Parameters
[in]vertexShaderDeclVertex declaration describing vertex program input parameters.
[in]vertexBufferDeclVertex declaration describing structure of a vertex buffer to be bound as input to the GPU program.
[in]vertexProgramInstance of the vertex program we are creating input layout for.
Note
Error will be thrown if the vertex buffer doesn't provide all the necessary data that the shader expects.
