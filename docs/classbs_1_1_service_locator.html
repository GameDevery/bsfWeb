Layers » Utility » General

Description

template<class T>
class bs::ServiceLocator< T >

A locator system that allows you to quickly find a service of a specific type.

Note
This is similar to a singleton pattern but unlike singleton the active instance is not required to be available and can be replaced with another system during runtime, or completely removed.
