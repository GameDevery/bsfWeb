Path Class Reference
Layers » Utility » File system

Description

Class for storing and manipulating file paths.

Paths may be parsed from and to raw strings according to various platform specific path types.

Note
In order to allow the system to easily distinguish between file and directory paths, try to ensure that all directory paths end with a separator (\ or / depending on platform). System won't fail if you don't but it will be easier to misuse.
