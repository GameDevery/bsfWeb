ResourceManifest Class Reference
Internals » Core » Resources

Description

Serializable class that contains UUID <-> file path mapping for resources.

Note

This class allows you to reference resources between sessions. At the end of a session save the resource manifest, and then restore it at the start of a new session. This way ensures that resource UUIDs stay consistent and anything referencing them can find the resources.
Thread safe.

Inherits IReflectable.
