RenderTexture Class Reference
Layers » Core » RenderAPI

Description

Render target specialization that allows you to render into one or multiple textures.

Such textures can then be used in other operations as GPU program input.

Note
Sim thread only. Retrieve core implementation from getCore() for core thread only functionality.

Inherits RenderTarget.

Inherited by D3D11RenderTexture, GLRenderTexture, and VulkanRenderTexture.
