◆ drawClearQuad()

void drawClearQuad ( UINT32  clearBuffers,
const Color &  color,
float  depth,
UINT16  stencil 
)

Draws a quad that clears the current viewport.

This is supposed to emulate functionality available with other APIs like DX9 and OpenGL where you can clear only a part of the render target. (DX11 API only provides a way to clear the entire render target).

Parameters
[in]clearBuffersCombination of one or more elements of FrameBufferType denoting which buffers are to be cleared.
[in]color(optional) The color to clear the color buffer with, if enabled.
[in]depth(optional) The value to initialize the depth buffer with, if enabled.
[in]stencil(optional) The value to initialize the stencil buffer with, if enabled.
