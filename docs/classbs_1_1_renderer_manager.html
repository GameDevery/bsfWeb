RendererManager Class Reference
Internals » Core » Renderer

Description

Allows you to change and retrieve the active renderer.

Active renderer will be used for rendering all objects in the following frame.

Note
No renderer is active by default. You must make a renderer active before doing any rendering.

Inherits Module< RendererManager >.
