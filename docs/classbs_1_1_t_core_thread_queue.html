Layers » Core » Core thread

Description

template<class CommandQueueSyncPolicy = CommandQueueNoSync>
class bs::TCoreThreadQueue< CommandQueueSyncPolicy >

Queue that allows the calling thread to queue commands for execution on the core thread.

Commands will only be executed after they have been submitted to the core thread.

Note
Queued commands are only executed after the call to submitToCoreThread(), in the order they were submitted.

Inherits CoreThreadQueueBase.
