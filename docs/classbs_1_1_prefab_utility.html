static void clearPrefabIds ( const HSceneObject &  sceneObject,
bool  recursive = true,
bool  clearRoot = true 
)
static

Clears all prefab "link" IDs in the provided object and its children.

Parameters
[in]sceneObjectPrefab instance to clear the link IDs from.
[in]recursiveIf true, all children of the provided scene object will be cleared as well.
[in]clearRootIf true, the root provided object will have its link ID cleared. If false the root object's components will have their IDs cleared but not the scene object itself. (Child scene objects will have their link IDs cleared in case recursive is true.)
Note
If any of its children belong to another prefab they will not be cleared.
