◆ createOcclusionQuery()

virtual SPtr<OcclusionQuery> createOcclusionQuery ( bool  binary,
UINT32  deviceIdx = 0 
) const
pure virtual

Creates a new occlusion query that allows you to know how many fragments were rendered between query start and end.

Parameters
[in]binaryIf query is binary it will not give you an exact count of fragments rendered, but will instead just return 0 (no fragments were rendered) or 1 (one or more fragments were rendered). Binary queries can return sooner as they potentially do not need to wait until all of the geometry is rendered.
[in]deviceIdxIndex of the GPU device to create the query on.

Implemented in VulkanQueryManager, D3D11QueryManager, and GLQueryManager.
